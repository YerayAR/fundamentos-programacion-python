# Fundamentos de Programación con Python

Un curso completo e interactivo de fundamentos de programación con Python, presentado como un sitio web moderno construido con Next.js.

## 🚀 Características

- **Contenido estructurado**: 10 módulos que cubren desde conceptos básicos hasta proyectos finales
- **Sitio web interactivo**: Interfaz moderna y responsiva con Next.js 14
- **Búsqueda integrada**: Encuentra rápidamente el contenido que necesitas
- **Modo oscuro**: Cambia entre temas claro y oscuro
- **Sintaxis destacada**: Código Python con resaltado de sintaxis

## 📚 Contenido del Curso

1. **Introducción** - Conceptos básicos y configuración del entorno
2. **Tipos de Datos** - Variables, números, strings y booleanos
3. **Control de Flujo** - Condicionales y bucles
4. **Estructuras de Datos** - Listas, tuplas, diccionarios y sets
5. **Funciones** - Definición, parámetros y scope
6. **Modularidad** - Módulos y paquetes
7. **Archivos** - Lectura y escritura de archivos
8. **Programación Orientada a Objetos** - Clases y objetos
9. **Testing** - Pruebas unitarias con pytest
10. **Proyecto Final** - Aplicación práctica de los conceptos

## 🛠️ Tecnologías

- [Next.js 14](https://nextjs.org/) - Framework de React
- [TypeScript](https://www.typescriptlang.org/) - Tipado estático
- [Tailwind CSS](https://tailwindcss.com/) - Estilos
- [MDX](https://mdxjs.com/) - Markdown con componentes React
- [MiniSearch](https://github.com/lucaong/minisearch) - Motor de búsqueda

## 🚀 Inicio Rápido

### Prerrequisitos

- Node.js 18+ 
- npm o yarn

### Instalación

```bash
# Instalar dependencias
cd curso-web
npm install

# Modo desarrollo
npm run dev

# O desde la raíz del proyecto
npm run dev
```

El sitio estará disponible en [http://localhost:3000](http://localhost:3000)

### Scripts Disponibles

```bash
npm run dev        # Inicia el servidor de desarrollo
npm run build      # Construye la aplicación para producción
npm start          # Inicia el servidor de producción
npm run lint       # Ejecuta ESLint
npm run typecheck  # Verifica tipos con TypeScript
```
